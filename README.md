# Controlador de Red

Aplicación de escritorio diseñada para supervisar dispositivos conectados a una red local en tiempo real. Permite identificar direcciones IP, direcciones MAC y fabricantes, además de asignar alias personalizados para reconocer cada dispositivo de forma más sencilla. También incluye un panel visual que facilita el monitoreo del estado de la red y de los equipos activos.

## Características

Controlador de Red ofrece una forma práctica de visualizar y gestionar los dispositivos conectados dentro de una red local. Entre sus funciones principales se encuentran la detección de equipos activos, la visualización de información de red relevante y la personalización de nombres para mejorar la organización y el reconocimiento de cada dispositivo.

## Funcionalidades principales

- Escaneo de dispositivos conectados en la red local.
- Identificación de direcciones IP de cada equipo detectado.
- Visualización de direcciones MAC.
- Detección del fabricante asociado a cada dispositivo.
- Asignación de alias personalizados.
- Panel visual para monitorear el estado de la red en tiempo real.
- Consulta rápida de dispositivos activos.

## Tecnologías utilizadas

Este proyecto fue desarrollado en Python y utiliza bibliotecas enfocadas en la creación de interfaces gráficas, escaneo de red y manejo de datos para ofrecer una experiencia visual clara y funcional.

## Objetivo del proyecto

El objetivo de esta aplicación es facilitar la supervisión de redes locales mediante una herramienta visual e intuitiva que permita identificar rápidamente los dispositivos conectados, organizar su información y mejorar el control general de la red.

## Posibles usos

Este sistema puede ser útil para tareas de monitoreo en hogares, oficinas, laboratorios o entornos educativos, donde sea importante conocer qué dispositivos se encuentran conectados a la red y mantener un control más ordenado de ellos.
